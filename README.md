## Overview
This project analyzes customer reviews from Zomato to understand user sentiment, rating distribution, and popular keywords. The goal is to extract meaningful insights from textual reviews using Python.

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- NLP (Text preprocessing)

## Dataset
- Source: Zomato reviews dataset (Kaggle)
- Format: CSV / TSV

## Key Features
- Data cleaning and preprocessing
- Sentiment analysis of customer reviews
- Rating distribution analysis
- Visualization of insights using graphs
- Modular Python code and Jupyter Notebook

## Visualizations
- Rating distribution bar charts
- Review length analysis
- Sentiment-wise comparison
